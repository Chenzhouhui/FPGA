# FPGA 学习记录

本仓库用于记录 FPGA 学习过程中的代码练习、语法笔记和项目总结，重点聚焦 Verilog HDL 的基础与工程实践。

## Verilog HDL 语法速览

### 1. 模块与端口
- `module ... endmodule`：模块定义的基本结构。
- 端口方向：`input`、`output`、`inout`。
- 端口类型：`wire`（连线型）、`reg`（过程赋值型，Verilog-2001 中常用于 `always` 块赋值）。
- 参数化：`parameter`、`localparam`，用于提高模块复用性。

### 2. 数据类型与位宽
- 标量与向量：`wire a;`、`reg [7:0] data;`
- 有符号/无符号：`signed`、默认无符号。
- 常量表示：`4'b1010`、`8'hFF`、`16'd255`。
- 特殊值：`x`（未知）、`z`（高阻）。

### 3. 运算符
- 算术：`+ - * / %`
- 关系：`> < >= <= == !=`
- 全等：`===`、`!==`（会比较 `x/z`）
- 逻辑：`&& || !`
- 按位：`& | ^ ~`
- 归约：`&a`、`|a`、`^a`
- 拼接与复制：`{a, b}`、`{4{a}}`
- 移位：`<<`、`>>`

### 4. 赋值语句
- 连续赋值：`assign y = a & b;`（驱动 `wire`）
- 阻塞赋值：`=`（组合逻辑常用）
- 非阻塞赋值：`<=`（时序逻辑常用）

### 5. 过程块
- `always @(*)`：描述组合逻辑。
- `always @(posedge clk)`：描述时序逻辑。
- `always @(posedge clk or negedge rst_n)`：带异步复位的时序逻辑。
- `initial`：仿真初始化（综合支持取决于器件和工具）。

### 6. 选择与循环
- 条件：`if / else`
- 多分支：`case / casex / casez`（`casex` 使用需谨慎）
- 循环：`for`、`while`、`repeat`、`forever`（后两者多用于仿真）
- 生成块：`generate ... endgenerate`

## Verilog 核心知识点

### 1. 组合逻辑与时序逻辑
- 组合逻辑输出只依赖当前输入；时序逻辑输出与时钟边沿相关。
- 组合逻辑应避免锁存器（`latch`）的意外推断。
- 时序逻辑建议统一使用非阻塞赋值 `<=`。

### 2. 时钟与复位设计
- 常见复位方式：同步复位、异步复位。
- 复位释放要注意时钟域同步，避免亚稳态。
- 时钟使能优先于门控时钟（除非有专门时钟资源支持）。

### 3. 状态机（FSM）
- 典型结构：状态寄存器、次态逻辑、输出逻辑。
- 编码方式：独热码、二进制码、格雷码。
- 建议采用“三段式”或“二段式”规范写法，提升可读性与可维护性。

### 4. 时序约束与收敛
- 关键指标：建立时间（setup）、保持时间（hold）、时钟偏斜（skew）。
- 约束文件（如 XDC）应完整定义时钟、输入输出延迟、跨时钟域路径。
- 关注关键路径，必要时进行流水线优化。
